Space Wars has been published

Published on 29 May 2020

SpaceWars is a simple and 2D game written in Python which can be used on PCs and Macs. I've just published on my GitHub the code. Feel free to read it and also to download the game.

Dependecies

The game needs Python 3 (I used the 3.8 version for developement so I reccomebd to install that version, which is the latest) and the pygame module. For install pygame you have to run the 'pip install pygame' command on your Terminal.
